This is a fast implementation of the python anesthesia simulator (PAS), using Myelin, a tool
for simplifying the process of creating physiological closed-loop controller pipelines.

Work is just starting, but I'm keeping the project open source from start to finish, and publishing under GPL3 license, just like PAS.
